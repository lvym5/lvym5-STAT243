R Commands

R Command Function Format
tally() Gives numerical results of a study tally(~variable, data=dataset)
head() Gives the first few rows and columns of the study head(dataset)
filter() Allows to filter through the data set filter(dataset, criterion)
gf_bar() Produces a bar graph gf_bar(~variable, data=dataset)
gf_histogram() Produces a histogram gf_histogram(~variable, data=dataset)
gf_density() Renders a "fluid" looking graph gf_density(~class, data=dataset)
mean() Gives the mean of the dataset mean(~variable, data=dataset)
median() Gives the median of the dataset median(~variable, data=dataset, na.rm = TRUE)
gf_dotplot() Produces a dotplot gf_dotplot(~variable, data=dataset, p = percentile)
gf_boxplot() Produces a boxplot gf_boxplot(~variable, data=dataset)
gf_point() Produces a scatterplot gf_point(~variable, data=dataset)
cor() Gives the correlation coefficient of the data set cor(y var~x var, data=dataset)
lm() Produces a linear model (can be piped to another graph using |>) lm(y var~ x var, data=dataset)
sample() Gives all numbers in random order sample(variable) such as creating a die <-(1:6) and then writing sample(die)
resample() Mimics a dice (can give multiples of different numbers) resample(variable, size=desired size)
do() Iterates a certain number of times do(desired number)
prop() Yields relative frequency for specific number prop(~(sample == desired number), data=dataset) or just prop(function == desired times to iterate)
trunc() Rounds numbers to non-decimal numbers trunc(number or equation)
gf_segment() Yields a simple graph gf_segment(y var ~ x var + x var)
dbinom() Singular probability for a pmf dbinom(x, size, probability)
pbinom() Accumulating probability for a cdf pbinom(x, size, probability)
dexp() Pdf for x that is exponential dexp(x, rate= λ)
pexp() Cdf for x that is exponential (antiderivative of dexp()) pexp(x, rate= λ)
qexp() Gives median quartile qexp(x, rate= λ)
rexp() Gives a random generation from the exponential distribution rexp(x, rate= λ)
rnorm() Generates n random numbers from a normal distribution rnorm(n, mean=mean, sd= standard deviation)
pnorm() Returns the cumulative probability of a standard normal distribution pnorm(x, mean, sd)
qnorm() Returns quantile qnorm(x, mean, sd)
rbinom() Gives the probability that x is a certain number rbinom(x, mean, pr)
qbinom() Finds the number p so that pr(x ≤ p) is the output qbinom(p, size, pr)
pbinom() The probability that p ≤ k, which indicates what quantile a score of k is at. Can be useful to find the area under wanted curve pbinom(x, size, pr) (or 1-pbinom(x, size, pr) and for a 2 sided test, multiply by 2)
gf_dist() Creates a distribution gf_dist("dist name", parameters)
favstats() Gives standard deviation, mean, max, etc. favstats(~variable, data=dataset)
t.test() Gives a one sample t-test or CI t.test(~variable, data=dataset) such as t.test(~variable, data = dataset, conf. level = 0.92)
pchisq() Gives the p-value from a chi-squared statistic pchisq(test stat, df)
chisq.test() Create a chi-squared table chisq.test(myTable)$ expected
anova() Gives the F-stat anova(lm(y var~xvar, data=dataset)
pf() Gives p value for ANOVA 1-pf(F stat, k, n)
TukeyHSD() Tells whether there is a significant different in pairs of means in a data set TukeyHSD(aov(y var~x var, data=dataset)
fitted() Extracts fitted values from the fitted model object fitted(lm(y var~xvar, data=dataset)
makeFun() Creates a function funcName <- makeFun(function)
plot() Plots distribution of positive and negative intervals plot(linear model, which=1)
