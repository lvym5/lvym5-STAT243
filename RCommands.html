R Commands

The following is a list of useful R commands. Hover over each command for its format and a relevant example.

tally()

Gives numerical results of a study

tally(~variable, data = dataset)
Ex: tally(~x_var, data = quant_Data)

head()

Gives the first few rows of the study

head(dataset)
Ex: head(quant_Data)

filter()

Allows to filter through the dataset

filter(dataset, criterion)
Ex: filter(quant_Data, age > 25)

gf_bar()

Produces a bar graph

gf_bar(~variable, data = dataset)
Ex: gf_bar(~x_var, data = quant_Data)

gf_histogram()

Produces a histogram

gf_histogram(~variable, data = dataset)
Ex: gf_histogram(~x_var, data = quant_Data)

gf_density()

Renders a "fluid" looking graph

gf_density(~class, data = dataset)
Ex: gf_density(~cars, data = quant_Data

mean()

Gives the mean of the dataset

mean(~variable, data = dataset)
Ex: mean(~x_var, data = quant_Data)

median()

Gives the median of the dataset

median(~variable, data = dataset, na.rm = TRUE)
Ex:median(~x_var, data = quant_DATA, na.rm = TRUE)

gf_dotplot()

Produces a dotplot

gf_dotplot(~variable, data = dataset, p = percentile)
Ex: gf_dotplot(~x_var, data = quant_Data, p = 0.35)

gf_boxplot()

Produces a boxplot

gf_boxplot(~variable, data=dataset)
Ex:gf_boxplot(~x_var, datat = quant_Data)

gf_point()

Produces a scatterplot

gf_point(~variable, data = dataset)
Ex: gf_point(~x_var, data = quant_Data)

cor()

Gives the correlation coefficient of the data

cor(yVar ~ xVar, data = dataset)
Ex: cor(y_var ~ x_var, data = quant_Data)

lm()

Produces a linear model to be piped to a graph

lm(yVar ~ xVar, data = dataset)
Ex: lm(y_var ~ x_var, data = quant_Data) (use |> to pipe to a graph)

sample()

Gives all numbers in random order

sample(variable_)
Ex: die <-(1:6)
sample(die)

resample()

Mimics a dice

resample(variable, size = desired size)
Ex: resample(x_var, size = 6)

do()

Iterates a certain number of times

do(desired number)
Ex: do(sample(1:6, size = 3, replace = TRUE))

prop()

Yields relative frequency for specific number

prop(~(sample == desired number), data = dataset)
Ex: prop(~(sample == 5), data = quant_Data)

trunc()

Rounds numbers to non-decimal numbers

trunc(number or equation)
Ex: trunc(15.86)
trunc(15*4.7)

gf_segment()

Yields a simple graph

gf_segment(yVar ~ xVar)
Ex: gf_segment(y_var ~ x_var)

dbinom()

Singular probability for a pmf

dbinom(x, size, probability)
Ex: debinom(successes, size = 5, prob = 0.5)

pbinom()

Accumulating probability for a cdf

pbinom(x, size, probability)
Ex: pbinom(successes, size = 4, prob = 0.2)

dexp()

pdf for x that is exponential

dexp(x, rate = λ)
Ex: dexp(value for pdf, rate = 0.5)

pexp()

cdf for x that is exponential (antiderivative of dexp)

pexp(x, rate = λ)
Ex: pexp(value for cdf, rate = 0.5)

qexp()

Gives median quartile

qexp(x, rate = λ)
Ex: qexp(probability value for computing quartile), rate = 0.75)

rexp()

Gives a random generation from the exponential distribution

rexp(x, rate = λ)
Ex: rexp(number of random numbers desired, rate = 0.5)

rnorm()

Generatets n random numbesr from a normal distribution

rnorm(n, mean=mean, sd = standard deviation)
Ex: rnorm(number of random numbers desired, mean = 20, sd = 3.2)

pnorm()

Returns the cumulative probabilty of a standard normal distribution

pnorm(number of random numbers desired mean = 20, sd = 3.2)
Ex:

qnorm()

Returns quantile

qnorm(x, mean, sd)
Ex: qnorm(probability value, mean = 10, sd = 1.1)

rbinom()

Gives the probability that x is a certain number

rbinom(x, mean, pr)
Ex: rbinom(numbers desired to generate, mean = 10, prob = 0.5)

qbinom()

Finds the number p so that pr(x ≤ p) is the output

qbinom(p, size, pr)
Ex: qbinom(probability value, size = 10, prob = 0.5)

pbinom()

The probability that p ≤ k, which indicates what quantile a score of k is at

pbinom(x, size, pr) or 1-pbinom(x, size, pr) for a two sided test, multiply by 2>
Ex: pbinom(successes, size = 5, prob = 0.5)

gf_dist()

Creates a distribution

gf_dist("dist name", parameters)
Ex: gf_dist("carCount", x > 5)

favstats()

Gives standard deviation, max, mean, etc

favstats(~variable, data = dataset)

t.test()

Gives a one sample t-test or CI

t.test(~variable, data = dataset)
Ex: t.test(~carCount, data = quant_Data, conf. level = 0.92)

pchisq()

Gives the p-value from a chi-squared statistic

pchisq(test stat, df)
Ex: pchisq(x_var, df = 4)

chisq.test()

Creates a chi-squared table

chisq.test(myTable)$ expected
Ex:

anova()

Provides the F-stat

anova(lm(yVar ~ xVar, data=dataset)
Ex: anova(lm(~carCount ~ bikeCount, data = quant_Data)

pf()

Gives the p-value for ANOVA

1-pf(F stat, degrees of freedom of numerator, degrees of freedom of denominator)
Ex: 1-pf(30.5, 2, 9)

TukeyHSD()

Tells whether there is a significant different in pairs of means in a data set

TukeyHSD(aov(yVar ~ xVar, data = dataset)
Ex: TukeyHSD(aov(carCount ~ bikeCount, data = quantData)

fitted()

Extracts fitted values from the fitted model object

fitted(lm(yVar ~ xVar, data = dataset)
Ex: fitted(lm(carCount ~ bikeCount, data = quant_Data)

makeFun()

Creates a function

funcName <- makeFun(function)
Ex: multiply_nums <- makeFun(x*y)

plot()

Plots distribution of positive and negative intervals

plot(linear model, which=1)
