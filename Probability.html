Probability

Probability: Long term relative frequency that an event occurs over repetition of the random process.

Events can be combined (algebra of events):

  • Supposed A,B are events, S is the sample space. A∈B means all outcomes in A are found in B
  • A∩B is A and B for those outcomes common to both
  • A∪B means A or B
  • When A∩B is 0, we say that A and B are mutually exclusive, because the probability of both of the events happening is 0.
  • Ac is known as the complement of A, or 1-A

    • Probability Rules

  • Probabilities must be between 0 and 1

  • Probability of anything happening in a sample space is 1

  • For disjoint events, A∪B = P(A) + P(B)

  • P(A) = number of outcomes in A/total number of outcomes

  • P(A∪B) = P(A) + P(B) - P(A∩B)

  • Conditional Probability P(B|A) = P(A∩B)/P(A)

  • Bayes Rule:
    P(A|B) = (P(B|A) * P(A)) / P(B)

  • X is a random variable if it is a numerical outcome of some random process

  • The population we draw from affects the sample distribution, so increase the sample size for better accuracy.

    • Random Variable calculations

    Mean μx, the expected value of x: ∑xx*Pr(X=x)

    Variance σ2x is computed as ∑x(x-μx)2*Pr(X=x)

    X probability for density f(x): Pr(a ≤ x ≤ b)) = ∫ab f(x)dx

  • Requirements of a pdf (Probability Density Function): never negative, total area under the curve is 1
  • If we set f(x) = ∫-∞xf(t)dt, we accumulate a cdf (Cumulative Distribution Function

    Binomial Variables
  • The population we draw from affects the sample distribution, so increase the sample size for better accuracy.

    • Instances of random binomial variables examples:
  • Take 10 free throws with a 70% accuracy shooter. Let X be the count of free throws made.
  • Draw with replacement, n = 5 times from a bag containing proportion 3/7 blue tokens where X is the count of blue tokens
    • Common features to identify a binomial setting:
  • Random process breaks into success/repetitive Bernoulli trials
  • These trials are carried out a predetermined number of times
  • 2 results (success or failure)
  • Trials are independent
  • Pr(success) is known number P, failure is 1-P

    • When X ~ Binom(n, p):

    • E(x) = μx = np
    • Var(x) = np(1 - p)
    • SD(x) = √(np(1 - p))
